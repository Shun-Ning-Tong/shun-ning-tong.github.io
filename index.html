順寧堂

源於1959年，
創始人黃昌憑著對山草藥的知識，
開始為筲箕灣街坊服務，
自行調制跌打藥，專治跌打筋傷，
沿用至今。

服務宗旨

都市人因運動損傷，
工作勞損，
深受痛症困擾。
本診所希望以
傳統及沒有副作用的
療法治療病人。

睇症三部曲

1 /

註冊中醫師診斷，推薦合適治療方法
( 正骨/針灸/推拿/外敷藥/內服藥 )，讓病人自己決定

2 /

註冊中醫師正骨 / 針灸 / 開藥 ;
專業推拿師推拿

3 /

註冊中醫師建議是否需要覆診，
及日後注意事項

三大優勢

1. 註冊中醫師診斷，絕不加重傷勢
2. 專業推拿師(男及女)，注重手法治療
3. 傳承袓傳跌打藥，沿用六十載，
加入防敏藥材

治療項目

跌打 (中醫骨傷)

手法治療，先捽後敷
直達患處，徹底根治

針灸治療

專業治療，無副作用
刺激患處，效果顯著
